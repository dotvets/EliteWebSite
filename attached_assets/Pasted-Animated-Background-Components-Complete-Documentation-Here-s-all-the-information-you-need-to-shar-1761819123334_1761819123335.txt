Animated Background Components - Complete Documentation
Here's all the information you need to share with the Replit Agent for implementing these animated backgrounds on another website:

Overview
The Dr. Paws website uses two animated ECG (electrocardiogram) heartbeat line animations that continuously scroll across the screen. Both use pure SVG animations (no JavaScript required).

Component 1: ECGAnimation.tsx
Purpose: Header animated background that appears at the top of pages
Location: Used in page headers (Home, About, Services, Blog, Book Now, Admin pages)

Full Component Code:
export default function ECGAnimation() {
  return (
    <div className="w-full overflow-hidden bg-background">
      <div className="relative h-20 md:h-24">
        <svg
          className="w-full h-full absolute inset-0"
          viewBox="0 0 1200 140"
          preserveAspectRatio="xMidYMid meet"
          xmlns="http://www.w3.org/2000/svg"
        >
          <defs>
            <pattern
              id="ecg-pattern-header"
              x="0"
              y="0"
              width="400"
              height="140"
              patternUnits="userSpaceOnUse"
            >
              <animateTransform
                attributeName="patternTransform"
                type="translate"
                from="0 0"
                to="400 0"
                dur="12s"
                repeatCount="indefinite"
              />
              <path
                className="ecg-path"
                d="M0,70 L60,70 L65,65 L70,70 L75,70 L80,70 L85,50 L90,90 L95,70 L100,70 L105,75 L110,70 L115,70 L400,70"
                fill="none"
                stroke="#18ac61"
                strokeWidth="4"
                strokeLinecap="round"
                strokeLinejoin="round"
              />
            </pattern>
          </defs>
          
          <rect
            x="-400"
            y="0"
            width="2000"
            height="140"
            fill="url(#ecg-pattern-header)"
          />
        </svg>
      </div>
      
      <style>{`
        @media (min-width: 768px) {
          .ecg-path {
            stroke-width: 2.5;
          }
        }
      `}</style>
    </div>
  );
}

How It Works
1. SVG Pattern Animation
Uses SVG <pattern> element to create a repeating heartbeat pattern
<animateTransform> animates the pattern from left to right infinitely
Pattern width: 400px, repeats seamlessly
2. Animation Details
Duration: 12 seconds per cycle (dur="12s")
Direction: Left to right (from="0 0" to="400 0")
Loop: Infinite (repeatCount="indefinite")
Type: Smooth linear translation
3. Heartbeat Path
The ECG line is drawn using SVG path coordinates:

d="M0,70 L60,70 L65,65 L70,70 L75,70 L80,70 L85,50 L90,90 L95,70 L100,70 L105,75 L110,70 L115,70 L400,70"

This creates the characteristic heartbeat spike pattern.

Customization Options
Change Color:
Modify the stroke attribute in the <path>:

stroke="#18ac61"  // Current green color

Change Speed:
Modify the dur attribute in <animateTransform>:

dur="12s"  // Slower
dur="6s"   // Faster

Change Height:
Mobile: h-20 (80px)
Desktop: h-24 (96px) for ECGAnimation
Desktop: h-36 (144px) for HeartbeatDivider

Change Line Thickness:
strokeWidth="4"  // Mobile
strokeWidth="2.5" // Desktop (in media query)

Notes for Implementation
No external dependencies needed - Pure HTML/SVG/CSS
Tailwind CSS classes used - Ensure Tailwind is configured
Pattern ID must be unique - Each component uses different IDs:
ECGAnimation: ecg-pattern-header
HeartbeatDivider: ecg-pattern
Color should match brand - Currently uses #18ac61 (Dr. Paws primary green)
